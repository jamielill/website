MoleSweeper | Project Type: Game Jam | Theme: Roots

Info

This project was created as part of the Global Game Jam 2023 and was my first ever game jam! I worked with 3 of my coursemates and we created a two player tile based top down game inspired by 'mine sweeper'. The aim is to dig up a mole before your opponent does. However you can also dig up different root vegetables such as the hot potato which explodes a 3x3 grid of tiles, and the beatroot which punches the player to the edge of the map.

My contribution

My main role was content integrator, ensuring that all assets were imported without compression as well as compiling animations together. I coded the grid generation system along with the hot potato mechanic.

Software used

Unity, Adobe Photoshop
