Spot-Billed Toucanet | Project Type: Model | Type: Character with Generic rig

This Model and rig was created with the intention of implementing it within a game and was developed in 12 weeks for a University module. The video shows the process and more WIP pictures can be found below.

What I learned

>>> How to create a generic rig from scratch within Blender
>>> How to create a control rig with appropriate custom bone shapes
>>> How to create animations that blend together

Potential improvements:

>>> Curve based wing controls
>>> A larger range of feathers to create more accurate wings
>>> Feather particle system baked into a texture
>>> Lower poly model

Blender, Unity
