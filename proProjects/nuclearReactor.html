Nuclear Reactor Panel | Project Type: Game | Type: co-op game with custom controller

This group project is a bit different. Our task was to create a custom controller as well as a game in unity to go along with it. Our group decided to make a nuclear reactor control panel puzzle game similar to "please don't touch anything" and "Keep talking and nobody explodes".

My contribution

my role was controller designer, I wired up the buttons and switches to an Arduino board and programmed it to send signals to Unity. I also laser-cut wooden panels to create the casing of the reactor control panel.

What I learned

>>> How to wire up buttons and switches to an Arduino board
>>> How to program an Arduino to send signals to Unity
>>> How to use a laser cutter to create the casing for the controller

Potential improvements:

>>> 3D printed the casing instead of laser cutting it
>>> Adding more puzzles to the game

Software used

Unity, Arduino IDE, Adobe Illustrator
