Boozer Blitz | Project Type: Game | Type: 3D Bartending Game

This is a solo project with all 3D assets created by me. Your role is a bartender that must serve drinks to life sized glasses of alcohol before they get 'pissed' and attack you. If this happens you must make shots and cocktails such as the Jager Bomb which will explode the enemies.

What I learned

>>> Transferring skills from Unity to Unreal Engine
>>> How to use Unreal Engine's Blueprints to create a game
>>> Creating AI agents in Unreal

Potential improvements:

>>> having more cocktail recipes
>>> Adding more enemies
>>> Adding more levels such as a nightclub

Software used
