Currents Album Cover

This is my attempt at "Currents" Album cover by Tame Impala and was created in August 2024
>>> V1.0 Created the basic design using a plane with a wave texture and a sphere
>>> V1.1 seperated the red line to a seperate plane below the waves as it makes controlling the gradient texture easier
>>> V1.2 Created a rough curve for the red line

Specs:
>>> Software: Blender
>>> Final Render: Cycles
>>> Tris: 4k
